# AI_Document_Analyzer
A Python-based intelligent document reader that uses OpenCV and Tesseract OCR to extract text from images, and integrates Hugging Face LLM APIs for text translation and summarization.

##Key Features:

- Extract text from scanned documents or images using OCR

- Translate extracted text into multiple languages

- Summarize long documents using state-of-the-art LLMs

- Modular architecture (OCR, Translation, Summarization)

- Uses Flask for a simple web interface

Tech Stack:
Python 路 OpenCV 路 Tesseract 路 Hugging Face 路 Flask
